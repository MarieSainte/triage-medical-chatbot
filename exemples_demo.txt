=========================================================
  MEDICAL CHATBOT — EXEMPLES DE DÉMONSTRATION (POC)
  Modèle : Qwen3-1.7B-Base + LoRA SFT + DPO
  Format de sortie : JSON structuré (type / urgence / analyse / question)
=========================================================

Les exemples ci-dessous illustrent les trois cas gérés par le modèle :
  - URGENCE HAUTE   : appel immédiat du 15 / SAMU
  - URGENCE MOYENNE : consultation rapide (< 24h)
  - URGENCE FAIBLE  : peut attendre, soins à domicile
  - QUESTION        : informations insuffisantes, le modèle demande des précisions

=========================================================
  NOTES TECHNIQUES
=========================================================
  - Endpoint API  : POST /triage/ask  { "symptomes": "..." }
  - Réponse status: "ANALYSE" (urgence + analyse) | "ASSISTANT" (question)
  - Modèle servi  : vLLM + LoRA adapter (SFT → DPO)
  - Prompt optimisé via DSPy BootstrapFewShot (4 demos, val_accuracy 100%)
